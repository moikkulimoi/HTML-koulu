<HTML>
<HTML> (HyperText Markup Language) on verkkosivujen perusrakennekieli.
CSS (Cascading Style Sheets) on tyylitiedostokieli, joka määrittelee, miltä HTML-elementit näyttävät.
CSS tyylit määritellään yleensä erillisessä .css-tiedostossa, joka linkitetään HTML-sivuun <head>-osiossa.
<head>
<head> sisältää taustatietoa (metadataa) verkkosivusta.
CSS muotoilun tiedosto reitti tulee tähän osioon. esim css/tyylit.css
<Body>
<body> sisältää kaiken näkyvän sisällön, joka näytetään käyttäjälle verkkosivulla.
CSS muotoilu <body> style="background-color: lightblue;"> </body> Tämä muuttaa taustavärin vaaleansiniseksi.
<p>
<p> tarkoittaa paragraph eli kappale. Sillä merkitään tavallista tekstisisältöä verkkosivulla.
CSS muotoilu <p style="color: red;"> Tämä teksti on punainen. </p> Tämä muuttaa tekstin värin punaiseksi.
<img>
<img>‑elementti lisää kuvan verkkosivulle. Se ei ole tekstielementti, vaan upottaa ulkoisen tai paikallisen kuvatiedoston sivulle.
CSS muotoilu <img src="Kuvat/html kuva.jpg" width="400" style="border: 2px solid black;"> Tämä lisää mustan kehyksen kuvan ympärille.
<a>
<a> (anchor) luo linkin, jota klikkaamalla käyttäjä siirtyy toiseen sivuun, tiedostoon tai samaan sivuun toiseen kohtaan. href="..." → osoite, johon linkki vie
CSS muotoilu <a href="https://www.google.com/" style="color: green;"> Avaa Google </a> Tämä muuttaa linkin värin vihreäksi.
<br>
<br> tekee rivinvaihdon tekstin sisällä — sama kuin ”enter” ilman uutta kappaletta.
CSS muotoilu <br> voi olla <br style="line-height: 2;"> Tämä lisää tilaa rivinvaihdon ympärille.
